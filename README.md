Travel Journal App

This project is my Mobile Development Milestone 1

Description
The Travel Journal App allows users to view, add, and manage their travel destinations.  
I built it using Kotlin and Jetpack Compose, with navigation between multiple composable screens.

Features
- Welcome Screen  
- Destinations Screen  
- Add Destination Screen  
- Data validation for input fields  
- Simple and modern Compose UI  

 Technologies i Used
- Android Studio  
- Kotlin  
- Jetpack Compose  
- GitHub for version control
